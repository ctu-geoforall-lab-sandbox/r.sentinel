NAME

r.sentinel - Toolset for download and processing of Sentinel products.

KEYWORDS

raster, import, Sentinel

DESCRIPTION

r.sentinel toolset consists of two modules:
r.sentinel.download
downloads Sentinel products from Copernicus Open Access Hub
r.sentinel.import
imports already downloaded Sentinel products into GRASS mapset

REQUIREMENTS

  • Sentinelsat library
  • Pandas library

AUTHOR

Martin Landa, GeoForAll Lab, CTU in Prague, Czech Republic with support of OpenGeoLabs company
